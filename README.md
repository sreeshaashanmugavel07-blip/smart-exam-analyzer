# 🚀 Smart Exam Analyzer

👉 **Live Demo:** https://your-streamlit-app-link.streamlit.app
👉 **Demo Video:** https://your-video-link.com

An AI-powered web app that helps students **analyze past exam papers**, identify **high-yield topics**, and generate a **smart study plan**.

---

## 🎯 Problem

Students rely on past question papers but:

* ❌ No clear way to find **important topics**
* ❌ Hard to identify **patterns across years**
* ❌ Waste time on **low-value topics**
* ❌ No structured **study planning**

---

## 💡 Solution

Smart Exam Analyzer uses AI to:

* Extract topics from past papers
* Identify trends and question patterns
* Rank high-priority topics
* Generate a **personalized study plan**

---

## ⚙️ How It Works

1. 📂 Upload PDFs or images of question papers
2. 🔍 Extract text (PDF + OCR)
3. 🧠 Analyze using Groq AI
4. 📊 Generate insights
5. 📅 Create a study plan

---

## 📊 Key Outputs

* 📌 Topic Frequency Analysis
* 🏆 Topic Importance Ranking
* 📈 Question Pattern Analysis
* 📚 Syllabus Coverage
* 🗓️ Smart Study Plan
* ✍️ Practice Questions
